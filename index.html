Salut, mon nom est Zachary, un développeur full-stack passionné avec un diplôme en intégration multimédia. Je prospère face aux défis, me dépassant toujours pour atteindre mes objectifs. Flickflow est une application dotée de fonctionnalités similaires à celles d'un gestionnaire de films : elle permet l'authentification des utilisateurs, l'ajout de leurs favoris, la possibilité de donner leur propre note à une série, ainsi qu'un moteur de recherche. Toutes les données proviennent d'une API externe RESTful.
